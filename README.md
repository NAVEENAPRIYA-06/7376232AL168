# Campus Notification System

## Frontend Track Submission

This project is developed for the AffordMed Campus Hiring Evaluation.

## Features

- Display all notifications
- Priority notifications page
- Filter notifications by type
- Pagination support
- Read/Unread notification status
- Responsive UI using Material UI
- Logging middleware integration

## Tech Stack

- React.js
- Material UI
- Node.js
- Express.js
- Axios

## Project Structure

notification_app_fe → React frontend

notification_app_be → Backend proxy server

logging-middleware → Logging middleware implementation
